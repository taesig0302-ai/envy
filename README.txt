ENVY — Season 1 (Dual Proxy Edition + Smartstore Crawler)
---------------------------------------------------------
실행 방법:
1. Python 3.10 이상 환경에서 아래 패키지를 설치하세요.
   pip install streamlit requests beautifulsoup4

2. 아래 명령어로 실행합니다.
   streamlit run app.py

3. 브라우저에서 "🧲 네이버 스마트스토어 크롤러" 섹션에 URL을 입력하면
   상품명, 가격, 이미지를 자동으로 불러옵니다.
